TourBookingWeb

A tour booking web application built with ASP.NET Core MVC.
This project allows users to browse tours, make bookings, and enables administrators to manage tour information.

FEATURES
- User authentication and authorization
- Tour listing and tour details
- Tour booking functionality
- Admin management for tours and bookings
- MVC architecture

TECH STACK
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript
- Razor Views

PROJECT STRUCTURE
TourBookingWeb/
- Controllers/
- Models/
- Views/
- Services/
- Data/
- Migrations/
- wwwroot/
- Program.cs
- appsettings.json
- TourBookingWeb.csproj

DATABASE
- SQL Server
- Entity Framework Core Migrations
- SQL schema file: database/schema.sql
- No sensitive data included

HOW TO RUN
1. Clone repository
2. Configure connection string in appsettings.json
3. Run database migrations
4. Run project using: dotnet run
